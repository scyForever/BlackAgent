# BlackAgent 交付文档

> 本文档按 `黑灰产情报分析Agent.docx` 的目标说明当前仓库已经交付的工程能力、产物、数字和边界。所有数字来自当前已复跑的本地产物；历史局部 run 不再作为主交付口径。

## 1. 交付结论

当前项目已经落成一条本地可复跑的公开 / 授权黑灰产情报分析流水线：

```text
公开/授权情报采集
  -> 多模态文本物化
  -> 清洗与去重
  -> 风险分类
  -> 实体抽取
  -> 实体图谱与线索质量评估
  -> 人工复核候选与报告
```

当前可声称本地公开 / 授权 manual held-out gold 已就绪；仍不声称线上生产泛化或自动化处置。联网采集、真实 LLM 和生产处置默认关闭或 dry-run。

## 9. 线索质量评估

线索质量评估不再只依赖分类 F1，`src/enhancement/clue_quality.py` 已加入：

- evidence count
- cross-source support
- critical entity support
- average classification confidence
- freshness score / reasons
- false-positive risk score / reasons

这使线索质量能同时表达证据链强度、跨源支撑、时效性和误报风险。

## 11. 安全与合规边界

| 边界 | 实现 |
| --- | --- |
| 默认不联网 | `config/config.yaml` 中 `network.enabled=false` |
| source 合法依据校验 | `src/safety/source_policy_guard.py` |
| 禁止登录/验证码/私群绕过 | source policy 硬规则 |
| URL 凭据泄露拦截 | token/cookie/session/basic auth 检查 |
| 默认 LLM dry-run | `llm.enabled=false`、`llm.dry_run=true` |
| PII masking | `src/safety/pii_masker.py` |
| 生产处置默认 dry-run | `enforcement.enabled=false`、`enforcement.dry_run=true` |

当前产物用于公开 / 授权防御情报分析，不用于绕过访问控制或自动化攻击。
