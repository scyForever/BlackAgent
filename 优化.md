 第 3 点：真实多源 300-500 条验收包
  已优化为无 r.jina.ai 依赖的公开源验收流程，避免原先 r.jina.ai HTTP 401 导致真实包为空。

  1. 数据源清单使用 config/intel_sources.acceptance_direct.yaml，覆盖四类：公众号/文章、二手交易、众包/问答、技术论坛。
  2. 401 源不再继续依赖 r.jina.ai，改为直连公开 Bing/DuckDuckGo HTML 搜索结果解析；登录 cookie/API token 只作为未来接入授权源时的合法方式，不用于当前验收包。
  3. 采集命令示例：
     python scripts/collect_public_sources.py --catalog config/intel_sources.acceptance_direct.yaml --db data/acceptance_direct_collect_final3.db --summary-out data/acceptance_direct_collect_final3_summary.json --max-records 20 --max-sources 230 --source-name-max-quota 18 --source-min-quota public_account_or_article=60 --source-min-quota secondhand_market=40 --source-min-quota crowdsourcing_platform=40 --source-min-quota vertical_or_technical=80 --rate-limit-per-minute 30 --retry-attempts 1 --retry-backoff-seconds 2 --fresh
  4. 导出命令示例：
     python scripts/export_delivery_corpora.py --db data/acceptance_direct_collect_final3.db --raw-jsonl-out data/acceptance_direct_final3_raw_dataset.jsonl --quota-jsonl-out data/acceptance_direct_final3_quota_sample.jsonl --defense-quota-jsonl-out data/acceptance_direct_final3_defense_quota_sample.jsonl --acceptance-pack-jsonl-out data/collection_phase_multi_source_acceptance_pack.jsonl --acceptance-pack-classifications data/acceptance_direct_final3_raw_classifications.jsonl --acceptance-pack-entities data/acceptance_direct_final3_raw_entities.jsonl --manifest-out data/acceptance_direct_final3_delivery_manifest.json
  5. 导出时启用 evidence-ready trace gate：验收包只从同时具备分类和实体证据的 trace 中抽样。本次 405 条原始记录中 401 条通过 gate，排除 4 条无实体噪声结果，仍满足四类候选余量。
  6. 当前验收结果：data/collection_phase_multi_source_acceptance_pack.jsonl 为 300 条，四类各 75 条；manifest 的 claim_boundary 为 completed_300_to_500_multi_source_acceptance_pack。
  7. 审查证据包：data/collection_phase_multi_source_evidence_pack.jsonl 为 300 条；每条都有 source_type、原文片段、可审计清洗文本、分类、实体、线索链状态。清洗来源为 cleaning_phase 217 条、evidence_pack_inline_cleaning 83 条；跨源线索当前均标记为 no_cross_source_clue_yet，不冒充已形成跨源线索。

  重点不是凑数量，而是能证明“不同来源都能稳定产出可审查证据”。当前仍需人工抽检 Bing/DuckDuckGo 搜索结果噪声；已不再把无实体记录纳入 300 条验收包。
